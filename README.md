# AI-Driven Smart Portfolio Generator

An AI-powered smart portfolio platform that enables users to create dynamic professional portfolios by entering their skills, projects, and achievements, with an integrated Groq API chatbot for intelligent content enhancement, auto-update functionality, and shareable live portfolio links.

## Features
- Dynamic portfolio generation
- Groq API chatbot integration
- Real-time content enhancement
- Auto-updating portfolio system
- Shareable live portfolio links
- User-friendly UI

## Tech Stack
- React.js
- Node.js
- Groq API
- HTML/CSS/JavaScript

## Future Enhancements
- Authentication system
- Multiple portfolio templates
- Resume export functionality
- AI-based portfolio scoring
